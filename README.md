# UBypass

A simple web app for checking ETA of bus/MTR near HKU campuses, originally created for the course `COMP3330 - Interactive Mobile Application Design and Programming` as a hybrid app. 

Features: 
1. Display ETA for specified bus stops/MTR stations
2. Display current temperature and issued weather warnings
3. Filter routes/destinations based on user provided condition
4. Sort ETA by route number/arrival time
5. Display traffic nearby and current traffic news

To use: 
1. Go to [UBypass](https://lainumero1.github.io/ubypass-public)
2. Information shall be displayed in seconds for stops near the main campus
3. Click on the boxes for bus stops to view all departures at the stop
4. Search for specific routes/destinations by typing in the search box on top
5. Click `≡` on top for more features
6. To check nearby traffic, click `≡`, select `Nearby Traffic`
7. To sort ETA in a different way, enable/disable dark mode, or view ETA of a different campus, click `≡`, select `Settings`
8. To refresh ETA, click the refresh icon on the top of the screen
